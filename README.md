#To-Do Tasks

A simple app to keep your to-do tasks lists

### Install

Just run:

```bash
bundle install
rake db:setup
rails s
```

### Try it
Having everything ready and running, go to `http://localhost:3000` and enjoy! :)

or use online version deployed on [Heroku](http://todo-linden.herokuapp.com/)
